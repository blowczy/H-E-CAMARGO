# Projeto de Automatização de Emails para o Departamento de Suprimentos da H&E CAMARGO

## Requisitos de Instalação

- Python 3.11
- Bibliotecas: `os`, `glob`, `pyautogui`, `time`, `pandas`, `pyperclip`

Você pode instalar as bibliotecas necessárias executando no shell:

pip install pandas pyautogui pyperclip


## Descrição

Este algoritmo foi desenvolvido para aumentar a produtividade 
do departamento de suprimentos da H&E CAMARGO. 

Ele automatiza o envio de e-mails para fornecedores 
com informações específicas extraídas de arquivos Excel.
Depois cria arquivos com o historico das informações do processo em .txt

## Utilização

1. Mantenha o Microsoft Outlook logado e na tela inicial ("Home").
2. Coloque os arquivos Excel que você deseja processar na pasta: 
"C:\Users\user\Desktop\cotações recebidas\".
3. Execute o script.

### Atenção

Certifique-se de que a pasta e os arquivos Excel não estejam 
sendo utilizados por outros programas ao executar o script.
